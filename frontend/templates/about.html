About Project

Dataset

Based on the UCI Heart Disease schema: each row is one patient with routine clinical features (age, chest pain, blood pressure, cholesterol, ECG, etc.).

The target is a binary label (heart disease: yes / no), and the data is preprocessed as a tabular classification problem.

Project models – RBM, FNN, LSTM

RBM (Restricted Boltzmann Machine) is used to learn hidden features from the input data in an unsupervised way, which can then support better heart-disease prediction.

FNN (Feedforward Neural Network) maps patient features through hidden layers to predict risk from the tabular heart-disease dataset.

LSTM (Long Short-Term Memory) can model time-based patterns when sequential or time-series medical data is available for each patient.

Real-world use

The system is intended as a decision-support tool that quickly summarizes heart-disease risk from routine clinical inputs to guide further medical evaluation.

In practice, such models can support screening dashboards in clinics, telemedicine triage, and patient education, always alongside clinician judgment.
